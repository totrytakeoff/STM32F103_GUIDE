# STM32F103C8T6 Datasheets

`STM32F103C8T6` belongs to the STM32F103x8 medium-density device family.

## Files

- `STM32F103x8_STM32F103xB_datasheet_DS5319.pdf`
  - English ST official datasheet.
  - Source: https://www.st.com/resource/en/datasheet/stm32f103rb.pdf
  - Verified as a 114-page PDF.

- `STM32F103x8_STM32F103xB_datasheet_DS5319_ZH.pdf`
  - Chinese datasheet for STM32F103x8 / STM32F103xB.
  - STMCU Chinese official page: https://www.stmcu.com.cn/Designresource/detail/document/696088
  - Downloaded from public mirror because the STMCU direct PDF endpoint returns a login/status HTML page in non-interactive download.
  - Mirror: https://hybrogen.github.io/images/20230320/STM32F103x8B_DS_CH_V10.pdf
  - Verified as a Chinese PDF by extracting the first pages.

## Note

The Chinese file is useful for reading concepts and pin/peripheral descriptions in Chinese, but it is older than the English official datasheet in this directory. For final electrical limits, ordering information, and revision-sensitive details, prefer the English ST official PDF.
